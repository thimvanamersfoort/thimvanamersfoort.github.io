Windows Retro

Een website gebaseerd op het oude Windows-98 besturingssysteem. Destijds gemaakt voor een schoolproject. Voor interactie met buttons, en het verslepen van de "applicaties" in de website hebben we veel gebruik gemaakt van JQuery.

Achterliggende technologie

De onderliggende code stelt in dit project niet veel voor. Voor ons ging het hier meer om de gebruikerservaring dan het neerzetten van iets innoverends qua code. Een paar hoogtepunten in de code:

  • Gebruik van cookies: Bij inloggen wordt er een cookie opgeslagen die de gebruikersnaam bevat. Als deze beschikbaar is bij het laden van de website, zal de gebruiker geen inlogscherm te zien krijgen.
  • File Explorer: Bij het gebruik van de File Explorer opent de gebruiker daadwerkelijk bestaande bestanden, beschikbaar op een achterliggende PHP-server. Met AJAX-requests wordt de inhoud van deze bestanden opgevraagd.
  • Muziekspeler: Bij het opvragen van MP3-bestanden van de server, kunnen deze in de Browser afgespeeld worden via de Audio WebAPI en een op maat gemaakte UI.
