HSO Hertzinger Password Generator

Een simpele wachtwoordgenerator, in opdracht van HSO Hertzinger. In nood voor een wachtwoordgenerator (bestaande uit willekeurige tekens, voor gebruik als standaard Wi-Fi wachtwoorden) is deze applicatie ontworpen.

De broncode zelf stelt weinig voor. Geschreven in C# en de .NET-Core. Gebruikersinput wordt verzameld wat voor soort wachtwoorden er gegenereerd moeten worden (zie 1e screenshot) en vervolgens worden er willekeurige tekens uit een Array gehaald om deze wachtwoorden samen te stellen.

De bijbehorende installer:

Om het gebruik voor de medewerkers van HSO Hertzinger makkelijker te maken, is er besloten een voorgeprogrammeerde installer bij te doen. De gebruikte installer hier is de Visual Studio Installer Setup Project (inbegrepen bij het .NET-Framework).

Hierbij is de standaardconfiguratie gebruikt, dus de broncode van de installer is niet inbegrepen in de Github-repo. Wel is het mogelijk om een zelf installer toe te voegen aan de Visual Studio Solution. Hiervoor heb ik deze handleiding gebruikt.
