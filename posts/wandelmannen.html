De Wandelmannen Blog

De Wandelmannen is een project uitgevoerd voor een oom van mij. Na het lopen van de Tour de Santiago de Compostella, wilde hij deze memorabele reis graag ergens beschrijven, voor familie en vrienden om te volgen terwijl hij deze 13-daagse route aan het lopen was.

Achterliggende code

De PHP-code is ingedeeld op een manier waarop het voor mij (en eventuele andere contributeurs) makkelijk schaalbaar en overzichtelijk zou blijven. De HTML-code is allemaal inbegrepen in de normale .php-bestanden, waar ook SSR op toegepast wordt. Extra helperklasses en functies zijn terug te vinden in de includes/*.inc.php. Ook is er een page-components/ folder, waar veelgebruikte paginaonderdelen bewaard worden (bijv. footers).

Vernuftige functies:

  • Inloggen via Facebook om automatisch een bericht te plaatsen op de tijdlijn, wanneer er een nieuwe blogpost gemaakt is.
  • Gebruikers kunnen (anoniem) comments achterlaten, deze worden opgeslagen als een JSON String in de database.
  • Een carousel op de indexpagina die uitgelichte / meest gelezen blogposts roteert.
